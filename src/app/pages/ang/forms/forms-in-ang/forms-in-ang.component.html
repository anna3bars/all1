now when we clicking on this submit button it's logging the html of that form but here instead of logging this form as a javascript object or we can say typescript object

for this to this local reference variable (#myForm) we assign a string called ngForm 

 
  onSubmit(form: NgForm){
   console.log(form);
   }
  // and we also need to import it 
  import { NgForm } from '@angular/forms';
 
 
  <form (ngSubmit)="onSubmit(myForm)" #myForm="ngForm"></form>

So here we are getting this form as an object but in that object we dont have these controls

for that we need to do 2 changes
  1. for each of these input elements we need to specify the name
  2. write ngModel in this input

this ngModel will simply tell angular that this element - this input element is a control of this form 

 
  <input type="text" id="fname" placeholder="Your name.." 
   name="firstname" ngModel>
