Anghular is one of the most popular JavaScript framework for building client side applications

Anfular is a JavaScript Framework which allows us to create a single page applications (SPA)

A single page application is a web application, which has only one html page. When we navigate around, only the content of that HTML page changes. The page itself never changes. And also the content is changed by JavaScript dynamically, whenever we are navigating around we are not requesting any data from the server, JavaScript is simply changing the content based on the resource which we have requested
