Hierarchical Dependency Injection

In a hierarchical dependency injection, same instance of a service class is injected in a component and its child components and their child components.

so for the header component we ate going to get an instance of subscribeService, let's call it subService, we are not using providers array in any other child component, so according to a herarchical Injection since we have used the provider's array on the header component an instance of the service will be injected in the header component and same instance will also get inject in all its child components and their child components
