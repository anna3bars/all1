How to pass query param to the route

query parameters are the optional (необязательный) parameters that we pass to a route and these query parameters are added at the end of the url separated by a question mark "?"

The different between query parameters and router parameters

  • O- query parameters are the optional (необязательный) parameters, if the querry parameter is missing in the url it will not stop angular from navigathing to the route
  • O- query parameters are not part of the route defination
