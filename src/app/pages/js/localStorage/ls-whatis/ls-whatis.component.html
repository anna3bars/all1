Local Storrage это локально хранилище, которая находиться в вашем браузере, вы там можете можете хранить любые данние и они будут доступны у вас после перезагрузки страницы

то есть это локальная и импровизирования база данных для именно вашего браузера

в node нет local storage, это только браузерный api, он доступен в обьекте window

чтобы что то получить из local storrage мы вызываем метод getItem()

для того чтобы записать что то в local storrage мы вызываем метод setItem(), которое принимает 2 значение

  1. первое значение -- это ключ по которому нужно хранить что то
  2. второй аргумент -- это само значение

local storage умеет работать ТОЛЬКО со СТРОКОМИ когда мы передаум не строковую значение он автоматно вызиыает .toString()
